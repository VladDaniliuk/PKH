# EPIC 02 — Authentication

## Goal

Implement the first authentication flow for Android and Desktop clients backed by the backend service.

## In scope

- backend auth model and endpoints
- token/session flow
- shared auth logic
- sign-in screens
- logout
- invalid session handling

## Tasks

- [ ] Implement backend user model and auth endpoints
- [ ] Implement token-based authentication flow
- [ ] Add shared auth domain models and use cases
- [ ] Persist auth session locally in shared layer
- [ ] Implement sign in screen on Android
- [ ] Implement sign in screen on Desktop
- [ ] Handle expired or invalid token gracefully
- [ ] Implement logout flow
