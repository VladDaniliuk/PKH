# Roadmap

## R1 — Capture + Library MVP
Goal: deliver the first end-to-end flow:
auth -> add URL -> processing -> library -> item details -> collections/tags

Included:
- Foundation & project setup
- Authentication
- Capture URL / Add Item
- Library & Collections

## R2 — Reader Experience
Goal: make saved content more useful to consume.

Included:
- Reader screen
- Reading progress
- Better content display
- Reader-oriented UX polish

## R3 — Notes, Highlights, Search
Goal: turn saved materials into reusable knowledge.

Included:
- Notes
- Highlights
- Search
- Better organization

## R4+
Possible directions:
- Sync/offline-first improvements
- Import/export
- Smart collections
- Resurfacing
- Knowledge graph
